# Открытые данные Кыргызстана

Публичный дашборд ключевых показателей страны из официальных открытых
источников. У каждой цифры — ссылка на источник и дата данных. Без мнений
и прогнозов.

Некоммерческий проект. Данные НСК КР распространяются по лицензии
CC BY-NC-SA — атрибуция обязательна.

## Разработка

```sh
npm install          # фронтенд (Astro + ECharts + Tailwind)
npm run dev          # dev-сервер

pip install -r requirements.txt   # ETL, Python 3.12
python -m pytest                  # тесты парсеров
```

Данные обновляются GitHub Actions по расписанию: ежедневные метрики —
02:00 UTC (07:00 Бишкек), месячная статистика НСК — отдельным джобом.

Подробнее об источниках и конвенциях — в [CLAUDE.md](CLAUDE.md);
методология — на странице `/methodology` сайта.
